## 0. Executive Summary

본 전략서는 AI 1인 IT 기업의 첫 번째 제품 "AgentDesk"의 전략적 방향을 수립한다. AgentDesk는 프리랜서와 1인 사업자가 AI 에이전트 팀을 손쉽게 구성하고 운영할 수 있는 SaaS 플랫폼이다.

**핵심 전략 요약**:
- **카테고리**: AI Agent Team Operations (신규 카테고리 창출)
- **1차 타겟**: 프리랜서/1인 사업자 (PLG, 셀프 온보딩)
- **가격**: Freemium + Usage-based 하이브리드 ($0 / $29 / $79 / $199)
- **기술 스택**: Next.js + FastAPI + Supabase + Claude API
- **6개월 목표**: 유료 고객 50명, MRR $2,500 달성
- **Go/No-Go 판단**: 3개월차 유료 전환 10명 미만 시 Pivot 검토

## 1. 제품 비전 & 미션

### 1-1. 원라인 비전

**한국어**: "누구나 AI 팀을 가질 수 있는 세상"
**영어**: "An AI team for everyone"

### 1-2. 미션 스테이트먼트

> AgentDesk는 1인 사업자와 소규모 팀이 AI 에이전트로 구성된 가상 팀을 구축하여,
> 대기업 수준의 업무 역량을 갖출 수 있도록 돕는 것을 사명으로 한다.
> 우리는 복잡한 AI 기술을 단순한 팀 운영 경험으로 전환하여,
> 기업 규모가 아닌 아이디어의 크기로 성공이 결정되는 세상을 만든다.

### 1-3. 핵심 가치 제안 (Value Proposition)

**Before AgentDesk** (현재 고객의 고통):
- ChatGPT, Claude 등 범용 AI를 쓰지만 매번 컨텍스트를 다시 설정해야 한다
- 업무별 프롬프트를 매번 새로 작성해야 하고 결과 품질이 일관되지 않는다
- 여러 AI 도구 구독료가 누적되어 SaaS 피로도가 높다
- AI를 쓰지만 "팀처럼" 체계적으로 운영할 방법이 없다

**After AgentDesk** (제품이 제공하는 해결):
- 역할별 AI 에이전트(마케터, 개발자, 디자이너 등)를 드래그 앤 드롭으로 구성
- 에이전트가 업무 맥락을 기억하고 일관된 품질로 산출물 생성
- 하나의 구독으로 마케팅, 개발, 운영 등 전 영역의 AI 팀 운영
- 에이전트 간 협업 워크플로우로 복잡한 업무도 자동 처리

**핵심 가치 공식**:
```
AgentDesk = (AI 에이전트 팀 구성) + (워크플로우 자동화) + (도메인 특화 템플릿)
         = "AI 에이전트 조직을 SaaS로 제공"
```

## 2. 제품 포지셔닝

### 2-1. 카테고리 정의

**새로운 카테고리: AI Agent Team Operations (AI ATO)**

기존 카테고리와의 차별점:

| 기존 카테고리 | 대표 제품 | AgentDesk 차별점 |
|--------------|----------|-----------------|
| AI 챗봇 빌더 | Chatbase, Botpress | 단일 에이전트가 아닌 **팀(다수 에이전트)** 운영 |
| 워크플로우 자동화 | Make, Zapier | 단순 트리거가 아닌 **AI 판단 기반** 워크플로우 |
| AI 코파일럿 | GitHub Copilot, Notion AI | 단일 도구 내장이 아닌 **독립 에이전트 팀** |
| AI 에이전트 프레임워크 | CrewAI, LangGraph | 개발자 전용이 아닌 **비개발자도 사용 가능** |

**포지셔닝 문장 (Positioning Statement)**:
> "AI 도구를 하나하나 쓰는 대신, AI 팀을 통째로 운영하세요"
> AgentDesk는 1인 사업자와 소규모 팀을 위한 AI 에이전트 팀 운영 플랫폼입니다.
> 템플릿 기반으로 마케팅, 개발, 운영 에이전트 팀을 즉시 구성하고,
> 에이전트 간 협업으로 복잡한 업무를 자동 완료합니다.

**AgentDesk의 Sweet Spot**: 높은 에이전트 팀 역량 + 쉬운 사용성
- Make/Relay 대비: 단순 자동화가 아닌 AI 판단 기반 팀 운영
- CrewAI/LangGraph 대비: 코드 작성 없이 비개발자도 사용 가능
- Chatbase/Botpress 대비: 단일 봇이 아닌 팀(다수 에이전트) 오케스트레이션

### 2-3. Unique Selling Point (USP) 3가지

**USP 1: "팀 템플릿" -- 클릭 한 번으로 AI 팀 구성**
- 마케팅 팀, 콘텐츠 팀, 개발 팀, 고객지원 팀 등 사전 구성된 에이전트 팀 템플릿
- 사용자는 템플릿을 선택하고 자사 맥락만 입력하면 즉시 운영 시작
- 경쟁 제품: 개별 에이전트를 하나씩 설정해야 함 (설정 시간 10배 차이)
- **근거**: 자사 71개 에이전트 조직 운영 경험에서 도출한 실전 팀 구조

**USP 2: "에이전트 협업" -- 에이전트끼리 대화하고 협력**
- 에이전트 A(리서처)가 조사한 내용을 에이전트 B(라이터)가 받아 콘텐츠 작성
- 에이전트 간 자동 핸드오프, 품질 검수(QA 에이전트), 최종 승인 워크플로우
- 사용자가 개입할 포인트를 직접 설정 (Human-in-the-Loop)
- **근거**: Multi-Agent Orchestration 문의 1,445% 급증, 시장 수요 검증됨

**USP 3: "성과 대시보드" -- AI 팀이 만든 결과를 한눈에**
- 각 에이전트가 처리한 작업량, 절약한 시간, 생성한 산출물 통계
- "이번 달 AgentDesk가 절약해준 시간: 42시간, 비용 환산: $2,100"
- Outcome-based 가치 증명이 내장되어 구독 유지율(Retention) 극대화
- **근거**: Outcome-based 모델 부상 트렌드, 2030년까지 엔터프라이즈 SaaS 지출 40%가 성과 기반으로 전환 예상

## 3. 비즈니스 모델

### 3-1. 수익 모델: Freemium + Usage-based 하이브리드

**가격 설계 원칙**:
1. 프리 티어로 진입 장벽 제거 (PLG)
2. 기본 구독료로 안정적 수익 기반 확보
3. 사용량 기반 과금으로 성장에 따른 업사이드 확보
4. 성과 가시화로 가격 정당성 자동 증명

**가격 티어 상세**:

| **연간 결제** | - | $24/월 ($288/년) | $66/월 ($792/년) | $166/월 ($1,992/년) |

- 1순위 타겟(프리랜서) WTP: $15~$50 --> Starter $29가 심리적 수용 범위 중간값
- 2순위 타겟(스타트업 CTO) WTP: $50~$200 --> Growth $79, Agency $199가 수용 범위
- 경쟁 벤치마크: Make $10.59~$100+, Relay $38+, Gumloop $37~$244
- Free 티어 제공으로 PLG 전환 파이프라인 확보 (업계 무료-유료 전환율 평균 2~5%)

### 3-2. Outcome-based 과금 요소

기본 구독 위에 성과 연동 과금을 선택적으로 제공하여, Outcome-based 트렌드에 대응한다.

**Phase 1 (출시~6개월): Usage-based 초과 과금**
- 월 작업 실행 한도 초과 시 건당 과금 ($0.02~$0.05)
- 측정 용이, 고객 예측 가능, 구현 단순

**Phase 2 (6~12개월): Outcome Proxy 과금 도입**
- 에이전트가 생성한 "완성된 산출물" 기준 과금
  - 예: 완성된 블로그 포스트 1건 = 5 크레딧
  - 예: 완성된 이메일 시퀀스 1세트 = 10 크레딧
  - 예: 리서치 보고서 1건 = 8 크레딧
- 크레딧 기반으로 성과와 비용을 연결

**Phase 3 (12개월+): True Outcome-based 실험**
- "절약된 시간" 또는 "생성된 매출"에 연동하는 성과 기반 과금 테스트
- 예: "AgentDesk로 절약된 시간 1시간당 $2" 모델 검토
- 주의: 성과 측정의 신뢰성 확보가 전제 조건

## 5. Porter's Five Forces 분석

| Force | 강도 | 분석 | 자사 대응 |
|-------|------|------|----------|
| **신규 진입자 위협** | 높음 | AI 에이전트 빌더 진입 장벽 낮음. LangGraph/CrewAI로 누구나 구축 가능 | 팀 템플릿/도메인 노하우로 전환 비용 형성 |
| **대체재 위협** | 높음 | ChatGPT, Claude 직접 사용, 오픈소스 프레임워크 | "팀 운영" 경험 차별화 (단일 도구 vs 팀 시스템) |
| **구매자 교섭력** | 중간 | 프리랜서/1인 사업자는 가격 민감, 전환 비용 낮음 | 데이터/워크플로우 축적으로 Lock-in, 성과 가시화로 가치 증명 |
| **공급자 교섭력** | 중간 | Anthropic, OpenAI 등 LLM 공급사 의존 | 멀티 LLM 지원, 오픈소스 폴백 (Llama, DeepSeek) |
| **기존 경쟁** | 낮음~중간 | "AI 에이전트 팀 운영" 정확한 경쟁자 부재. 인접 경쟁만 존재 | 신규 카테고리 선점, 빠른 시장 교육 |

**종합 판단**: 산업 매력도는 **중상**이다. 진입 장벽이 낮아 경쟁자 출현이 빠를 수 있으나, "팀 운영 경험"이라는 차별화 축과 도메인 노하우 축적으로 지속 가능한 경쟁 우위를 확보할 수 있다. 핵심은 속도이다. 6개월 내 카테고리 선점이 전략적으로 결정적이다.
